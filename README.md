## How I Think About Agentic Systems

- **AI as teammate, not toy** — writer agents, reviewer agents, and monitoring on all of them
- **Security and observability first** — defence-in-depth, explicit threat models, metrics + traces + cost dashboards before ergonomics
- **Human-in-the-loop by design** — owners get plain-English verdicts; engineers get structured, traceable evidence
- **Observable everything** — if it's not in Grafana, it didn't happen
